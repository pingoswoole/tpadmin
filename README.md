ThinkPHP 6 + Swoole4
===============

> 运行环境要求PHP7.2+。

## 主要新特性

* 采用`PHP7`强类型（严格模式）
* 支持更多的`PSR`规范
* 原生多应用支持
* 更强大和易用的查询
* 全新的事件系统
* 模型事件和数据库事件统一纳入事件系统
* 模板引擎分离出核心
* 内部功能中间件化
* SESSION/Cookie机制改进
* 对Swoole以及协程支持改进
* 对IDE更加友好
* 统一和精简大量用法

## 安装

~~~
composer create-project pingo/tpadmin tpadmin
~~~
如果需要更新框架使用
~~~
composer update pingo/tpadmin
~~~
